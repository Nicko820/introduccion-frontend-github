# Muro de la Fama UBB: Charla Frontend & Git

¡Bienvenidos al repositorio oficial de la charla interactiva!

Este proyecto es una aplicación construida con **React** y **Vite**. El objetivo de esta dinámica es llevar la teoría a la práctica: vamos a construir un "Muro de la Fama" colaborativo donde cada uno de ustedes agregará su propia tarjeta al sitio web utilizando el flujo de trabajo profesional de **Git y GitHub**.

Al participar, pondrás en práctica conceptos clave como control de versiones, creación de archivos en repositorios remotos y la apertura de un **Pull Request (PR)**.

---

## ¿Cómo participar? (Paso a paso)

No necesitas instalar nada en tu computadora ni usar la terminal de comandos. Puedes hacer todo directamente desde la página web de GitHub:

1. **Navega a la carpeta de estudiantes:** En los archivos de arriba, haz clic para entrar a la carpeta llamada `estudiantes`.
2. **Crea tu archivo:** Haz clic en el botón **`Add file`** (arriba a la derecha de la lista de archivos) y selecciona **`Create new file`**.
3. **Nombra tu archivo:** En el recuadro del nombre (donde dice *Name your file...*), escribe tu nombre de usuario de GitHub seguido de la extensión `.json`. 
   > *Ejemplo: `navello.json`*
4. **Agrega tu información:** Copia el código JSON que está más abajo, pégalo en el editor de texto de GitHub y reemplaza los datos con los tuyos. ¡Cuidado con borrar las comillas o las comas!
5. **Guarda los cambios (Commit):** Haz clic en el botón verde **`Commit changes...`** (arriba a la derecha). En la ventana que se abre, escribe un mensaje corto como *"Agregando a [Tu Nombre] al muro"* y haz clic nuevamente en `Commit changes`.
6. **Abre el Pull Request:** GitHub detectará que hiciste un cambio y te sugerirá crear un PR. Sigue los pasos y haz clic en **`Create Pull Request`**.

¡Y listo! Yo revisaré tu solicitud en vivo y, al aprobarla (Merge), tu tarjeta aparecerá automáticamente en la pantalla de la presentación.
